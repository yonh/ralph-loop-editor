      • 
          
    • 要求自我修复：测试失败要定位原因、修复并重试，不通过不能跳步。
      •

    实现 Todo REST API（Node.js）。

完成标准：
- CRUD 接口全部可用并通过输入校验
- 测试覆盖率 >= 80%，lint 无错误
- README 包含启动方式与接口示例

执行流程：
1. 先写失败测试
2. 最小实现让测试通过
3. 运行测试与 lint
4. 失败则定位并修复后重试
5. 通过后再进入下一项

若连续 12 轮仍未完成：
- 输出阻塞原因
- 列出已尝试方案
- 给出建议的人工介入点

全部完成后输出：<promise>阶段工作结束</promise>
